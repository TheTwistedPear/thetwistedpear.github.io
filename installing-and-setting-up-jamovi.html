Chapter 13 Installing and Setting up Jamovi

Jamovi is a free and open source graphical statistical package. It is often described as “SPSS done properly”

We will be using Jamovi a lot in Psychological Measurement, and so this chapter will take you through how to install it and some of the additional modules that we will be using.

13.1 Installing Jamovi

You can download the latest version of Jamovi from jamovi.org. Click for link to Jamovi Downloads page

The website helpfully detects your operating system and will show you the appropriate downloads when you visit the site.

You should always opt for the ‘solid’ version, since ‘current’ will have experimental features and may be less stable.

13.1.1 Windows

You should just be able to download a .exe file and follow the installer instructions.

13.1.2 Mac

First you should determine whether your Mac has an Intel processor or an Apple Silicon processor.
Any Mac that was purchased new before 2020 will likely have an Intel processor; but you can check by selecting “About This Mac” from the Apple menu. If the chip name starts with an M, e.g. ‘Apple M3’, then you should download the Apple Silicon version of Jamovi.

Once you have downloaded the dmg file, double click on it and you will be presented with a Jamovi icon and an Applications folder Icon. To install Jamovi, click and drag the Jamovi icon into the Applications folder. If you do not complete this step then it may cause issues when saving your work.

13.1.3 ChromeOS/ Linux

You can install Jamovi on Linux and ChromeOS using flatpak or flathub respectively. For installing using flathub you can access the Jamovi page from the following link: Click for link to Jamovi Flathub page.

For more information on installation, please see the Jamovi user manual which can be found at the following linkL: Click for link to Jamovi user manual.

13.2 Installing Modules

In Psychological Measurement we will be using some additional modules for some of the tasks. To install these, load Jamovi and select theJamovi Library option from the Modules menu on the top right of the screen.

Next, locate and install the following modules (you can use the search function if you don’t want to scroll):

  • snowIRT
  • cNORMj
  • surveymv
  • seolmatrix
  • DiagROC

Installing these will add some extra options to the Jamovi analysis menu which will be covered in the relevant how-to guides.

13.3 Getting Started

Some of you will have already used Jamovi in a previous module. If you are new to Jamovi, or want to refresh your memory, then there is a playlist from the developers available at the following link: Click here for link to Jamovi playlist. As we move through the module, you will gain access to additional videos that are tailored to the workshops and tasks that are involved in the assessment.
